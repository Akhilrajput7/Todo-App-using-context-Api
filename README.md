A simple and responsive To-Do application built using React.js that helps users manage daily tasks efficiently.

link - https://gleaming-youtiao-3a3f61.netlify.app/

🚀 Features
Add new tasks
Delete tasks
Mark tasks as completed
Real-time UI updates
Responsive design
Clean and user-friendly interface

🛠️ Tech Stack
React.js
JavaScript (ES6+)
HTML5
CSS3 / Bootstrap

📂 Folder Structure
/src – Main source code
/components – Reusable UI components
/assets – Images / styles
App.js – Main component

⚙️ Installation & Setup
git clone https://github.com/Akhilrajput7/Todo-App-using-context-Api/
cd todo-app
npm install
npm start

💡 Usage
Enter a task in the input field
Click on "Add" button
Mark tasks as completed or delete them
